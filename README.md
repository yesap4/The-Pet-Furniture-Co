# The Pet Furniture Co

A static frontend landing page for a pet furniture brand built with React, Vite, and Tailwind CSS.

## Project Overview

This project is a UI showcase designed to present a pet furniture store experience. It includes a responsive menu, hero section, featured products, customer reviews, ratings, and a footer.

## Features

- Responsive React UI built with Vite
- Tailwind CSS styling and utility-based layout
- Hero banner with featured product highlight
- Navigation menu with search, cart, and profile controls
- Product review cards, ratings, and testimonials

## Tech Stack

- React
- Vite
- Tailwind CSS
- lucide-react icons
- ESLint


## Project Structure

- `src/App.jsx` — main app entry
- `src/Components/Menu` — top navigation and controls
- `src/Components/Hero Section` — hero banner and product highlight
- `src/Components/Footer` — footer content

## Getting Started

### Install dependencies

```bash
pnpm install
```

### Run development server

```bash
pnpm dev
```

### Build for production

```bash
pnpm build
```

### Preview production build

```bash
pnpm preview
```

## Notes

- This is a frontend UI project and does not include backend or checkout functionality.
- Ideal as a portfolio demo for React and Tailwind CSS.
